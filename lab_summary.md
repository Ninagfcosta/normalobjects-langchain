In this lab, the LangChain agent used tools in a freeform, creative way: 
when given a complaint, it independently decided which tools to call and 
in what order. This flexibility is great for open-ended creative tasks 
where the best path is not known in advance. Compared to a structured 
approach like LangGraph, which enforces a strict workflow, the freeform 
agent is more entertaining but less reliable for tasks requiring 
guaranteed steps. I would use LangChain for brainstorming or storytelling, 
and LangGraph when I need consistent, repeatable behavior.
